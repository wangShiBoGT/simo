## 基础功能测试

### WiFi AP 模式
- [ ] 手机能搜索到 `Simo-Robot` 热点
- [ ] 能用密码 `simo1234` 连接

### Web 控制页面
- [ ] 访问 `http://192.168.4.1` 能看到控制界面
- [ ] 界面样式正常显示（深色背景、蓝色按钮）

### API 测试
- [ ] `/ping` 返回 `PONG`
- [ ] `/info` 返回 JSON（芯片信息、版本等）

## STM32 通信测试（需连接硬件）

### 串口通信
- [ ] ESP32-S3 能发送命令到 STM32
- [ ] STM32 能返回 PONG 响应
- [ ] 传感器数据能正确解析

### 命令协议
- [ ] `PING` → `PONG`
- [ ] `M,F,150,500\n` → 前进
- [ ] `M,B,150,500\n` → 后退
- [ ] `M,L,150,500\n` → 左转
- [ ] `M,R,150,500\n` → 右转
- [ ] `STOP\n` → 停止
- [ ] `SENSOR\n` → 返回传感器数据
