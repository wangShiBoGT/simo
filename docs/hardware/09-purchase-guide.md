## 📝 卖家询价模板

### 模板 1：主板询价

```
你好，请问这块主板：

1. 是否有 PCIe x16 插槽？能装标准显卡（如 GTX1080）吗？
2. 内存插槽是几条？支持最大多少GB？
3. 有几个 SATA 接口？几个 M.2 接口？
4. 能拍一下主板实物照片吗？
5. 有没有测试过？能正常开机进 BIOS 吗？

谢谢！
```

### 模板 2：CPU + 主板套装询价

```
你好，请问这套 CPU + 主板：

1. CPU 是几代的？具体型号是什么？
2. 主板型号是什么？支持 DDR4 内存吗？
3. 有 PCIe x16 显卡插槽吗？
4. 是否带散热器？
5. 能正常开机使用吗？用了多久？
6. 包邮吗？

谢谢！
```

### 模板 3：深度相机询价

```
你好，请问这个 RealSense 相机：

1. 具体型号是 D435 还是 D435i？
2. 深度图像是否正常？有没有噪点？
3. USB 接口是否完好？
4. 能拍一段测试视频吗？（用官方 Viewer 软件）
5. 有原装线缆吗？
6. 使用了多久？为什么出？

谢谢！
```

### 模板 4：激光雷达询价

## ⚠️ 避坑指南

### 闲鱼购买注意事项

1. **看评价**：选择有交易记录的卖家
2. **要实拍**：不要相信网图，要求拍实物
3. **要测试视频**：关键部件（雷达、相机）要求运行视频
4. **确认配件**：问清楚是否包含线缆、电源等
5. **走平台**：不要私下转账
6. **保留聊天记录**：万一有问题可以维权

### 常见陷阱

| 陷阱 | 识别方法 |
|------|----------|
| **矿卡** | 显卡价格异常低，询问来源 |
| **翻新件** | 主板螺丝孔有划痕，问是否拆过 |
| **版本混淆** | D435 ≠ D435i，A1 ≠ A2 |
| **缺配件** | 明确问"包含哪些配件" |
| **运损** | 要求泡沫包装，保价快递 |
