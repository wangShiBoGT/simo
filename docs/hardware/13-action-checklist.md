## 📋 Phase 1：x86 主机（第 1-2 周）

### 目标：搭建一个能跑 Simo 后端的 Linux 主机

**预算：¥700-1200**

#### Step 1.1：采购主机零件

```
□ 1. 在闲鱼搜索 "i5 9400 + B365M 套装" 或类似
□ 2. 询问卖家：
     - 能正常开机吗？
     - 有几个 USB 口？
     - 支持 DDR4 内存吗？
□ 3. 确认无误后下单

□ 4. 同时购买：
     □ 内存 DDR4 8G（¥80-100）
     □ SSD 128G NVMe（¥60-80）
     □ 电源 400W（¥80-100）
     □ 机箱（¥50-80，可选先不买）
```

#### Step 1.2：组装主机

```
□ 1. 收到零件后检查外观
□ 2. 安装 CPU（如果是散装）
□ 3. 安装内存
□ 4. 安装 SSD
□ 5. 连接电源
□ 6. 裸机点亮测试（不装机箱）
```

#### Step 1.3：安装系统

```
□ 1. 下载 Ubuntu Server 22.04 LTS ISO
□ 2. 用 Rufus 制作 USB 启动盘
□ 3. 安装 Ubuntu（选择 minimal 安装）
□ 4. 配置网络（有线或 WiFi）
□ 5. 更新系统：sudo apt update && sudo apt upgrade
```

#### Step 1.4：部署 Simo 后端

```
□ 1. 安装 Node.js：
     curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
     sudo apt install -y nodejs

□ 2. 克隆 Simo 项目：
     git clone https://github.com/你的仓库/simo.git

□ 3. 安装依赖：
     cd simo && npm install

□ 4. 运行测试：
     npm run dev
```

#### Step 1.5：验收

```
□ 1. Simo 后端能启动
□ 2. 能通过浏览器访问 Web UI
□ 3. 主机运行稳定（跑 1 小时不死机）

✅ Phase 1 完成！
```

## 📋 Phase 2：STM32 连接（第 2-3 周）

### 目标：x86 主机能通过 USB 串口控制 STM32 小车

**预算：¥0（使用现有设备）**

#### Step 2.1：连接 STM32

```
□ 1. 用 USB 线连接 STM32 到 x86 主机
□ 2. 检查串口设备：
     ls /dev/ttyUSB*
     # 或
     ls /dev/ttyACM*

□ 3. 设置串口权限：
     sudo usermod -a -G dialout $USER
     # 重新登录生效
```

#### Step 2.2：测试通信

```
□ 1. 用 minicom 或 screen 测试串口：
     sudo apt install minicom
     minicom -D /dev/ttyUSB0 -b 115200

□ 2. 发送测试命令给 STM32
□ 3. 确认 STM32 有响应
```

#### Step 2.3：运行 Simo 控制

```
□ 1. 配置 Simo 的串口路径
□ 2. 启动 Simo 后端
□ 3. 通过 Web UI 控制小车移动
□ 4. 测试前进/后退/转向
```

#### Step 2.4：验收

```
□ 1. x86 主机能控制 STM32 小车
□ 2. 前进/后退/转向都正常
□ 3. 传感器数据能读取

✅ Phase 2 完成！
```

## 📋 Phase 3：语音交互（第 3-4 周）

### 目标：机器人能听、能说

**预算：¥400-500**

#### Step 3.1：采购音频设备

```
□ 1. 购买 ReSpeaker 麦克风阵列（¥350-400）
     淘宝搜索 "ReSpeaker XVF3800" 或 "ReSpeaker 2-Mics"

□ 2. 购买小扬声器 + 功放板（¥50）
     淘宝搜索 "PAM8403 功放 扬声器套件"
```

#### Step 3.2：安装麦克风

```
□ 1. 插入 USB 麦克风
□ 2. 检查识别：
     arecord -l

□ 3. 测试录音：
     arecord -D plughw:1,0 -f cd test.wav -d 5
     aplay test.wav
```

#### Step 3.3：安装扬声器

```
□ 1. 连接扬声器到 3.5mm 音频口
□ 2. 测试播放：
     speaker-test -c 2

□ 3. 或者用 USB 声卡
```

#### Step 3.4：集成语音 AI

```
□ 1. 配置语音识别 API（讯飞/阿里/OpenAI Whisper）
□ 2. 配置语音合成 API（讯飞/阿里/Edge TTS）
□ 3. 测试语音对话：说一句话 → AI 回复 → 播放
```

#### Step 3.5：验收

```
□ 1. 能对着麦克风说话
□ 2. 机器人能识别语音
□ 3. 机器人能语音回复

✅ Phase 3 完成！
```

## 📋 Phase 5：移动底盘（第 5-7 周）

### 目标：把主机装到底盘上，能移动

**预算：¥500-800**

#### Step 5.1：采购底盘

```
□ 1. 测量 x86 主机尺寸（长×宽×高）
□ 2. 淘宝搜索 "机器人 底盘 四轮 ROS"
□ 3. 确认：
     □ 尺寸 ≥ 25×25cm
     □ 承重 ≥ 5kg
     □ 带上层平台
     □ 带编码器电机（可选）
```

#### Step 5.2：采购电池

```
□ 1. 淘宝搜索 "3S 18650 电池组 11.1V"
□ 2. 容量 ≥ 5000mAh
□ 3. 带 BMS 保护板
□ 4. 带充电器
```

#### Step 5.3：组装底盘

```
□ 1. 组装底盘机械结构
□ 2. 安装电机
□ 3. 连接电机驱动板（L298N）
□ 4. 连接 STM32
□ 5. 连接电池
□ 6. 测试底盘能跑（用 STM32 单独控制）
```

#### Step 5.4：安装主机

```
□ 1. 在底盘上层平台固定主机
□ 2. 用铝型材/支柱搭建结构
□ 3. 理线：电源、USB 串口、HDMI
□ 4. 确保重心稳定（不会头重脚轻）
```

#### Step 5.5：集成测试

```
□ 1. 启动主机
□ 2. 连接 STM32
□ 3. 通过 Web UI 控制底盘移动
□ 4. 测试前进/后退/转向
```

#### Step 5.6：验收

