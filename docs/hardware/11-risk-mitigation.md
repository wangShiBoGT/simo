> **你的担心是对的。硬件不像软件，买错了就是真金白银。**
> 
> 这份文档帮你把每一分钱花对地方。

## 🚨 你的核心顾虑

1. **买错了** → 型号不对、接口不兼容
2. **买大了** → 尺寸不匹配、装不进底盘
3. **装不成** → 最后一堆零件躺在抽屉里
4. **浪费钱** → 每一笔都是真金白银

**我的策略：把风险分散到每一步，每一步都有"止损点"。**

### 第四步：加移动底盘（+¥500-700）

**目标**：把主机装到底盘上

**⚠️ 这一步最容易买错！**

```
购买前必须确认：
├── 底盘尺寸够不够放主机？（至少 25×25cm）
├── 底盘承重够不够？（至少 5kg）
├── 底盘有没有上层平台？（放主机用）
└── 电机接口和你的驱动板兼容吗？
```

**推荐策略：先量尺寸再买**

1. 量一下你的 x86 主机尺寸
2. 在淘宝找底盘时，问卖家要尺寸图
3. 确认能放下再下单

**购买清单**：
```
├── 四轮底盘套件（带电机+驱动）  ¥300-400
├── 3S 电池组                    ¥150-200
└── 支柱/连接件（铝型材）        ¥50-100

总计：约 ¥500-700
```

**验收标准**：
- [ ] 底盘能跑（前进后退转向）
- [ ] 主机能稳定放在底盘上
- [ ] 不会头重脚轻

**如果失败**：
- 底盘太小 → 买更大的底盘（损失当前底盘钱）
- 主机太重 → 换更小的主机或加配重

**止损点**：底盘是通用件，不合适可以闲鱼卖掉，通常能回 70%

## 📋 "买错了"防护清单

### 主板买错了

| 常见错误 | 如何避免 |
|----------|----------|
| 没有 PCIe 插槽 | 问卖家"有没有 PCIe x16" |
| 内存不兼容 | 问卖家"支持 DDR4 还是 DDR3" |
| 尺寸太大 | ITX < mATX < ATX，买 ITX 或 mATX |

### 底盘买错了

| 常见错误 | 如何避免 |
|----------|----------|
| 太小放不下主机 | 先量主机尺寸，买 ≥25cm 的底盘 |
| 承重不够 | 问卖家"最大承重多少" |
| 电机接口不对 | 确认是不是标准直流电机 |

### 深度相机买错了

| 常见错误 | 如何避免 |
|----------|----------|
| D415 ≠ D435 | D435 视角更广，买 D435 或 D435i |
| USB2.0 口不够 | 需要 USB3.0，确认主机有 USB3 口 |
| 二手有暗病 | 要求卖家发测试视频 |

### 激光雷达买错了

| 常见错误 | 如何避免 |
|----------|----------|
| A1 ≠ A2 | A1 够用且便宜，不需要 A2 |
| 转不起来 | 要求卖家发运行视频 |
| 供电不足 | 准备独立 5V 供电 |

## 🎯 给你的终极建议

### 1. 不要一次性买齐

```
错误：一次买 ¥6000 的东西
正确：先买 ¥1000 → 验证 → 再买 ¥500 → 验证 → ...
```

### 2. 每一步都要能"回头"

```
每买一批，问自己：
- 如果这批东西不能用，我损失多少？
- 这些东西能不能单独卖掉？
- 如果卡住了，我能不能暂停？
```

### 3. 关键件优先保品质

```
可以省钱的：机箱、支架、线材
不能省的：CPU、主板、深度相机
```

### 4. 利用 AI 做购买决策

```
买之前问我：
- "这个主板能不能用？"
- "这个底盘尺寸够吗？"
- "这个价格合理吗？"

我帮你判断
```
